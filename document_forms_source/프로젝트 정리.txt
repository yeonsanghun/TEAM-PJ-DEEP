[README.txt] 서식 문서 다중 라벨 분류 시스템 개발 프로젝트
1. 프로젝트 개요 (Project Overview)
본 프로젝트는 행정 서식(PDF) 데이터를 기반으로 특정 항목의 기재 여부나 오류를 자동으로 감지하는 딥러닝 기반 다중 라벨 분류(Multi-label Classification) 시스템 구축을 목표로 합니다. ConvNeXt 모델을 활용하며, 데이터 전처리부터 모델 최적화, 시각적 분석(Grad-CAM)까지의 전 과정을 포함합니다.

2. 주요 워크플로우 (Workflow)
Step 1: 데이터 전처리 (Data Preprocessing)
PDF 정제: 다중 페이지 PDF에서 분석 대상인 첫 번째 페이지만 추출합니다.

이미지 변환: 학습을 위해 PDF를 고해상도 JPG 이미지(300 DPI)로 변환합니다.

폴더 구조화: 파일명에 포함된 라벨 번호를 기반으로 데이터를 자동 분류합니다.

Step 2: 데이터셋 구축 (Dataset Preparation)
데이터 분할: 전체 데이터를 Train / Validation / Test 세트로 분리합니다.

라벨링: 각 이미지의 클래스 정보를 포함하는 labels.csv 파일을 생성합니다.

Step 3: 모델 학습 및 최적화 (Model Training & Optimization)
모델 설계: ConvNeXt Small 백본을 사용하여 14개의 클래스를 분류하는 헤드를 구성합니다.

최적화: Optuna를 사용하여 최적의 Learning Rate와 Batch Size를 탐색합니다.

Step 4: 성능 평가 및 분석 (Inference & Analysis)
추론: 학습된 체크포인트(.pth)를 로드하여 테스트 데이터의 정확도를 측정합니다.

시각화: Grad-CAM을 통해 모델이 이미지의 어느 영역을 보고 라벨을 분류했는지 히트맵으로 확인합니다.

3. 파일별 상세 설명 (File Descriptions)
[데이터 관리 및 전처리]
pdf_to_1pagepdf.ipynb: pypdf를 사용하여 모든 PDF의 첫 페이지만 남기고 저장합니다.

pdf_to_jpg.ipynb: pdf2image를 활용해 PDF를 학습용 이미지 파일로 변환합니다.

pdf_to_label_folder.ipynb: 파일명의 특정 규칙(예: 009-01)을 분석하여 해당 번호 폴더로 자동 이동시킵니다.

filename_change_by_foldername.ipynb: 경로 내 공백 제거 및 파일명 일괄 변경 등 정제 작업을 수행합니다.

train,val,test_split.ipynb: 파일명 기반(특정 번호대)으로 데이터를 훈련/검증/테스트 용도로 물리적 이동합니다.

generate_train_val_test_labels_csv.ipynb: 각 데이터 폴더를 스캔하여 모델 학습 시 입력값으로 쓰일 labels.csv를 자동 생성합니다.

[모델 학습 및 실험]
create_models_data2.ipynb:

기본 학습 스크립트.

이미지 크기 224x224, ConvNeXt Small 모델 사용.

RandomRotation, ColorJitter 등 데이터 증강 적용.

create_models_data3.ipynb:

실험 버전. 고해상도 입력(512x768) 및 다른 증강 기법을 적용하여 성능 변화를 확인합니다.

create_models_data2-optuna.ipynb:

Optuna를 연동하여 하이퍼파라미터 튜닝을 자동화합니다.

목적 함수(Objective) 설정을 통해 최적의 조합을 찾습니다.

[평가 및 분석]
load_models_data2.ipynb:

최종 모델 로드 및 추론 수행.

Grad-CAM 기술을 적용하여 서식 내의 어떤 항목(텍스트, 서명란 등)이 분류에 결정적이었는지 시각화합니다.

4. 환경 설정 (Environment & Dependencies)
Language: Python 3.x

Deep Learning: PyTorch, Torchvision

Data Analysis: Pandas, Numpy

Computer Vision: OpenCV, Pillow (PIL), pdf2image

Optimization: Optuna

Visualization: Matplotlib, pytorch-grad-cam

5. 사용 시 주의사항
경로 설정: 각 노트북 파일 상단의 root_dir 또는 MODEL_PATH 변수를 본인의 로컬 환경에 맞게 수정해야 합니다.

장치: CUDA 가속이 가능한 GPU 환경에서의 실행을 권장합니다. (코드 내 device = 'cuda' 설정 포함)

DPI: pdf_to_jpg 변환 시 고해상도(300 DPI)를 유지해야 서식 내 미세한 텍스트 식별이 가능합니다.
