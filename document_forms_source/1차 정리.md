[데이터셋 안내서] 문서 서식 필드 탐지 데이터셋 (Document Form Field Detection)1. 데이터셋 개요 (Dataset Overview)본 데이터셋은 문서 서식 이미지 내에서 항목(필드)의 오기입 또는 누락 여부를 탐지하는 딥러닝 모델을 학습시키기 위해 설계되었습니다. 전체 파이프라인은 PDF 변환, 라벨링, 데이터 분할, 그리고 ConvNeXt 모델을 이용한 멀티 라벨 분류 학습 과정을 포함합니다.2. 데이터 생성 파이프라인 (Data Generation Pipeline)Step 1: PDF 이미지 변환pdf2image 라이브러리를 사용하여 원본 PDF 문서를 이미지로 변환합니다.해상도(DPI): 300포맷: JPG변환 방식: 각 페이지를 개별 이미지 파일로 생성3. 라벨 정의 및 인코딩 (Label Definitions)본 데이터셋은 총 14개의 필드를 탐지 대상으로 하며, 멀티 라벨 분류(Multi-label Classification) 방식을 채택합니다.라벨 리스트 (14개 항목)ID필드명ID필드명01전입자_성명08현_연락처02전입자_주민등록번호09현_서명도장03전입자_연락처10현_주소04전입자_서명도장11전입사유_체크05전_시도12우편물서비스_동의체크06전_시군구13신청인_성명07현_세대주성명14신청인_서명도장인코딩 규칙 (Label Encoding)각 필드에 대해 이진값(Binary)을 부여합니다.1 (Correct): 해당 필드가 정상적으로 기입됨0 (Error): 해당 필드가 누락되었거나 오기입됨4. 데이터셋 분할 및 구조 (Dataset Split)파일명의 접두사(Prefix)인 문서 ID를 기준으로 데이터셋을 분리하며, 구조는 아래와 같습니다.Test: 문서 ID 005, 010Validation: 문서 ID 009Train: 그 외 모든 문서 ID디렉토리 구조 (Directory Structure)Plaintextdata2/

5. 모델 아키텍처 및 학습 설정 (Model & Training)모델 정보Architecture: ConvNeXt SmallInput Size: 224 x 224Method: Transfer Learning (Pretrained weights 사용)Classifier: Linear(768 → 14) 커스텀 레이어로 교체Loss Function: BCEWithLogitsLoss (멀티 라벨 분류 최적화)데이터 증강 (Data Augmentation)Resize (224x224)Random Rotation / Random Affine / Random PerspectiveColor Jitter (밝기, 대비 조절)Normalization: * Mean: [0.485, 0.456, 0.406]Std: [0.229, 0.224, 0.225]하이퍼파라미터 (Training Settings)Batch size: 8Optimizer: Adam (Learning rate: 1e-4)Epochs: 200Early stopping: 3 epochs (성능 개선 없을 시 조기 종료)6. 평가 지표 (Evaluation)모델의 최종 출력값은 시그모이드(Sigmoid) 함수를 통해 확률값으로 변환된 후 임계치(0.5)를 적용하여 결정됩니다.예측 로직$$\text{Probability} = \sigma(\text{logits})$$$$\text{Prediction} = \begin{cases} 1 & \text{if } \text{Probability} > 0.5 \\ 0 & \text{otherwise} \end{cases}$$주요 성능 지표 (Metrics)모델의 다각적인 성능 검증을 위해 다음과 같은 지표를 활용하고 출력합니다.정확도 (Accuracy): 전체 라벨에 대한 예측 정확도F1 Score (Micro/Macro/Weighted): 정밀도와 재현율의 조화 평균mAP (mean Average Precision): 모든 클래스에 대한 Average Precision의 평균값출력 예시:Pythonprint(f"정확도 : {acc:.4f}")
print(f"F1 micro : {f1_micro:.4f}")
print(f"F1 macro : {f1_macro:.4f}")
print(f"F1 weighted : {f1_weighted:.4f}")
print(f"mAP : {mAP:.4f}")
